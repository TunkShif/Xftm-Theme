### Installation
---

#### Requirements

- `pacman -S gtk-engine-murrine` *Arch Linux*
- `apt-get install gtk2-engines-murrine` *Debian*
- `dnf install gtk-murrine-engine` *Fedora*
- `zypper in gtk2-engine-murrine` *openSUSE*
- `emerge gtk-engines-murrine` *Gentoo*
